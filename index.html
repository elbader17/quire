Transforma hojas de cálculo en bases de datos relacionales para tus aplicaciones en Go. Sin configuración de servidor. Sin migraciones. Solo código.

$ go get github.com/elbader17/quire
main.go
package main

import (
    "github.com/elbader17/quire"
)

// 1. Define tu estructura de datos
type User struct {
    ID    int    `quire:"id"`
    Name  string `quire:"name"`
    Email string `quire:"email"`
}

func main() {
    // 2. Conecta con Sheets como si fuera SQL
    db, _ := quire.New(config)
    
    // 3. Query Builder fluido
    var results []User
    db.Table("Users").Query().
       Where("Name", "contains", "Alice").
       Limit(10).
       Get(ctx, &results)
}

Why Quire?

Lightning Fast Setup

Olvídate de configurar Postgres o MySQL para proyectos pequeños. Crea una Service Account, comparte el Sheet y estás listo.

Type Safety First

Si cambia en tu struct, Quire lo sabe. Validación de tipos fuerte en tiempo de compilación para evitar errores en runtime.

Developer Centric

API diseñada para humanos. Métodos encadenables, contexts, timeouts y manejo de errores idiomático de Go.

Read Rows.

El mapeo es automático. Solo agrega los tags `quire` a tus structs y la librería se encarga de convertir strings, números y booleanos mágicamente.

products.go
type Product struct {
    ID          int     `quire:"id"`
    Name        string  `quire:"product_name"`
    Price       float64 `quire:"price_usd"`
    IsAvailable bool    `quire:"in_stock"`
    secretCode  string  `quire:"-"` // Ignorado
}

// Así de simple es insertar:
func AddProduct(db *quire.DB) {
    item := Product{
        ID: 101, 
        Name: "Gaming Mouse", 
        Price: 59.99,
    }
    db.Table("Inventory").Insert(ctx, []Product{item})
}
